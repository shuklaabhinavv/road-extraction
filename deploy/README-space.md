# roadx

Extract road networks from live satellite imagery, anywhere on Earth, or from
your own aerial photos. Four segmentation models (U-Net, U-Net++, DeepLabV3+,
LinkNet) trained under an identical protocol on the Massachusetts Roads
Dataset.

Running on free CPU hardware — an extraction takes ~20–60 s.
